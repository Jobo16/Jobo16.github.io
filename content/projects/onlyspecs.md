# OnlySpecs

OnlySpecs 是一个 Electron 桌面应用，用于管理软件规格文档和对应实现版本。

## 主要能力

- Monaco Editor 多标签编辑。
- 多终端支持。
- 规格版本和代码版本管理。
- GitHub 仓库导入与分析。
- Claude AI 辅助生成和分析规格。

它关注的是一个问题：如果 AI 可以写代码，那么人应该怎样更好地维护“想做什么”和“已经实现了什么”之间的关系。
