# skills-kit

skills-kit 是一个面向 Codex skills 的仓库，用来沉淀可复用的工作流规则。

## 当前 skills

- `git-commit-doc`
- `goal-driven`
- `project-env-manager`

## 我为什么做它

AI Coding 的效果很依赖上下文和规则。把常用流程写成 skill，可以让 Agent 在不同项目中更稳定地执行提交说明、目标拆解、环境检查等任务。
