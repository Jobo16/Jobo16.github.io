# 小红书评论校验同步 Bot

这个项目用于把飞书多维表格中的小红书链接自动处理成结果表记录：读取待处理链接，抓取评论区内容，判断是否命中指定关键词，然后把结果批量写回飞书。

## 主要能力

- 飞书多维表格读取和写回。
- 小红书 Cookie 管理与有效性校验。
- 定时任务和手动同步。
- 失败重试和连续失败标记。
- 控制面板配置 Cookie 和同步状态。
- Docker 部署与持久化状态数据库。

## 项目价值

它把“人工打开链接、检查评论、复制结果”的重复流程变成了可追踪的自动化任务。对我来说，这也是一次把浏览器数据采集、飞书 API、后台任务和部署监控串起来的实践。
