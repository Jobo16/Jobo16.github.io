# 考情采集与分析工具

这个项目基于小红书内容采集能力，扩展出面向考情信息整理的批处理流程。

## 主要能力

- 批量读取 URL 列表。
- 抓取帖子标题、正文、评论和图片。
- 调用 OCR 与分析模型处理图片和文本。
- 导出结构化 Excel。
- 提供简单 Web 页面，支持手动粘贴 URL 并下载结果。

## 适用场景

把分散在平台内容里的考试经验、题型反馈和评论线索整理成可检索、可分析的结构化材料。
