# Jobo

这里会整理我正在做的事情、写下的文章、分享的内容、项目记录和联系方式。

## 现在

从 [[now|正在做什么]] 开始看。

## 内容入口

- [[writing|文章]]
- [[sharing|分享]]
- [[projects|项目]]
- [[about|关于]]
- [[contact|联系]]

## 维护方式

这个网站以 Obsidian 中的 Markdown 文件夹作为内容源，由 Quartz 渲染成静态网站。日常只需要维护本地 Markdown，推送到 GitHub 后自动构建上线。
