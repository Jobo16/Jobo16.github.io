# Blog Now Page Update

Use this skill when updating `content/now.md`.

## Goal

Keep the public Now page accurate, concise, and safe to publish.

## Required Sections

`content/now.md` should include:

- 更新时间
- 当前主要项目
- 最近关注
- 最近分享
- 最近阅读和学习

## Rules

- Write only information that is safe for public display.
- Do not include private schedules, private travel, private meetings, sensitive clients, internal roadmaps, or unconfirmed commitments.
- Prefer current priorities and themes over daily logs.
- Keep the page short enough to scan quickly.
- Update the visible date whenever the content is materially changed.
- If unsure whether something can be public, omit it or keep the page as `draft: true`.
