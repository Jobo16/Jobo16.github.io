# Project Skills

These project-level skills define how agents should maintain this Quartz + Obsidian blog.

Use these skills when editing content, syncing Obsidian files, preparing publishable Markdown, or reviewing public notes for privacy risks.

Current site structure:

- `content/index.md`
- `content/about.md`
- `content/contact.md`
- `content/projects/index.md`
- `content/projects/*.md`
- `content/notes/index.md`
- `content/notes/*.md`
