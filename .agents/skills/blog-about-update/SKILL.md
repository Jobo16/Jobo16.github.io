# Blog About Page Update

Use this skill when updating `content/about.md`.

## Goal

Keep the public About page accurate, concise, and safe to publish. This page owns the personal introduction, public contact links, and ongoing project snapshot.

## Required Sections

`content/about.md` should include:

- 简短个人介绍
- 联系
- Ongoing
- 我在做的事
- 背景
- 技能关键词
- 长期方向

## Rules

- Write only information that is safe for public display.
- Do not include private schedules, private travel, private meetings, sensitive clients, internal roadmaps, or unconfirmed commitments.
- Public contact links are allowed; private phone numbers, private email addresses, and private QR codes should not be added unless the user explicitly asks.
- Keep Ongoing focused on current public projects and link to `content/projects/*.md` pages where possible.
- Prefer current priorities and themes over daily logs.
- If unsure whether something can be public, omit it or keep the page as `draft: true`.
